🚀 En Güncel Teknolojilerin Kullanıldığı Restoran Yönetim Sistemi

Murat Yücedağ hocamın hazırlamış olduğu API Proje Kampı eğitiminden faydalanarak kapsamlı bir Asp.Net Core uygulaması geliştirdim.

Backend dünyasındaki en son yenilikleri takip ederek, .NET 10 platformu üzerinde inşa ettiğim modern restoran yönetim sistemi ve API projesini tamamlamanın mutluluğunu yaşıyorum! 

Bu projede, sadece bir CRUD uygulamasının ötesine geçerek; SignalR ile anlık veri akışı ve AI Entegrasyonu gibi akıllı çözümleri mimariye dahil ettim.

🏗️ Teknik Mimari ve Kullanılan Teknolojiler:
🔹 Backend: ASP.NET Core 10 Web API
🔹 Frontend: ASP.NET Core MVC
🔹 Database: SQL Server & Entity Framework Core
🔹 Design Pattern: Generic Repository Pattern
🔹 Real-Time Communication: SignalR ChatHub
🔹 AI Entegrasyonu: Yapay zeka destekli akıllı tarif oluşturucu ve otomatik mesaj yanıtlama modülleri
🔹 Mapping & Validation: AutoMapper (DTO yönetimi) & FluentValidation
🔹 API Doc: Swagger / OpenApi Entegrasyonu

🎯 Projenin Bana Kattıkları: Özellikle .NET 10'un getirdiği performans iyileştirmelerini deneyimleme, API ve UI projelerini birbirinden bağımsız şekilde yönetme ve yapay zeka servislerini bir iş akışına entegre etme konularında önemli tecrübeler edindim.
